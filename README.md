 🏚 Airbnb data scraping & property price prediction

- Developed web scraping pipeline using Selenium to extract 500+ Airbnb listings data
- Collected and stored structured data (pricing, location, reviews, features) in MongoDB
- Performed data cleaning, preprocessing, and feature engineering on real-world dataset
- Conducted exploratory data analysis and built visualizations using Streamlit framework
- Implemented regression model to predict property prices based on listing features
